# Visual-Basic-for-Applications---Excel-

"Simple things if done repetitively can really frustate you as a human
So let your "Smart" machine handle it" - Author: ME

I have created this repo with custom visual basic short scripts I commonly use in my day to day operations.
Each short script has a comment at top that defines its usecase.
Simple modifications like  changes to a row number or column character can alter the script to better suit your own needs.

Warning - The code once run cannot undo the changes.
So better learn how to version control your files when scripting.
A smart advice, from my personal frustating experiences.
So save the after script file as workbook1, workbook2 and so on.

Happy Scripting!
